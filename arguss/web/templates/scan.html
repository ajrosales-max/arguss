Scan a GitHub repo

Point Arguss at any public GitHub repository. Read-only analysis, no credentials required.

When to use this mode

Most common path. Use this when the repo is public and you want a quick read of its dependency health, including vulnerability findings, trust signals, and pipeline reality.

Expected URL format: https://github.com/{owner}/{repo}

Public GitHub repository, https URL.

Defaults to the repository's default branch.
