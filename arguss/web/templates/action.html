Scan with action

Autonomous remediation for repos you maintain. Arguss opens PRs for upgrades it can safely auto-merge.

When to use this mode

Use this when you want autonomous remediation, not just a scan. Arguss opens pull requests for the upgrades it can safely auto-merge and leaves the rest for human review. Best for repos you actively maintain. The PAT is used once and never stored.

The repo Arguss will scan and (where appropriate) open PRs against.

Defaults to the repository's default branch.

Arguss never stores your PAT. It's used to authenticate this single scan and discarded when the request completes.

Required PAT scopes

  • Contents read & write to read your lockfile and propose PR commits
  • Pull requests read & write to open and update PRs for safe upgrades
  • Actions read to inspect your CI workflows for the pipeline lens
